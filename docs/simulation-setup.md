# Simulation Environment Setup

This document outlines the setup process for various simulation environments used in the textbook.

## Gazebo

1. Install ROS 2 (Humble Hawksbill or later)
2. Install Gazebo Garden or Fortress
3. Set up the required packages:
   ```bash
   sudo apt install ros-humble-gazebo-ros-pkgs ros-humble-gazebo-ros2-control
   ```

## Unity

1. Download and install Unity Hub
2. Install Unity 2022.3 LTS or later
3. Install the required packages via Package Manager

## Isaac Sim

1. Install Isaac Sim from NVIDIA Developer website
2. Follow the installation guide for your platform
3. Set up the required dependencies
