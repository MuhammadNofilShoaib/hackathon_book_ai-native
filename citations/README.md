# Citations

This directory contains APA-style citations for the textbook content.

## Format

Each citation should follow the APA format:

```
Author, A. A. (Year). Title of work. Publisher. DOI or URL if applicable
```

## Examples

```
Russell, S., & Norvig, P. (2020). Artificial intelligence: A modern approach (4th ed.). Pearson.

Fox, R., & Long, D. (2003). The automatic inference of state invariants in TIM. Journal of Artificial Intelligence Research, 20, 321-360. https://doi.org/10.1613/jair.1185
```
